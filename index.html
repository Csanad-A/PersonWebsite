Year 2 Project
As a developer, I created a website similar to Reddit by first setting up a database to store user information, posts, and comments. I then designed the user interface using HTML, CSS, and JavaScript, allowing users to create an account, submit posts, and leave comments on others' posts. I implemented features such as upvoting and downvoting, as well as the ability to create and join communities. I also added moderation tools for administrators to manage the content on the site. To make the website functional, I wrote server-side code using a web framework such as Python's Django or Ruby on Rails to handle HTTP requests and interact with the database. Overall, creating a website like Reddit required a combination of front-end design and back-end development skills.
